## 一、项目概述

**项目名：** 鉴真仓（H5 + PC管理后台）

## 技术栈详情

### 用户端 H5

| 技术 | 选择 | 理由 |
|------|------|------|
| **框架** | Vue 3 (Composition API) | 简单易学，生态好 |
| **构建工具** | Vite | 快，现代 |
| **CSS框架** | TailwindCSS | 快速开发，体积小 |
| **PWA** | vite-plugin-pwa | 一键添加到桌面 |
| **路由** | Vue Router 4 | Vue 官方 |
| **HTTP客户端** | Axios | 通用 |

### 管理后台 PC

| 技术 | 选择 | 理由 |
|------|------|------|
| **框架** | Vue 3 | 与H5一致，降低学习成本 |
| **UI组件库** | Element Plus | Vue3生态最成熟 |
| **构建工具** | Vite | 快 |
| **状态管理** | Pinia | Vue3官方推荐 |

### 后端

| 技术 | 选择 | 理由 |
|------|------|------|
| **语言** | Node.js | 学习曲线低，Mattaniah可能熟悉 |
| **框架** | Express | 简单灵活 |
| **ORM** | Prisma | 现代化，易用 |
| **数据库** | SQLite（开发）/ PostgreSQL（生产） | SQLite免部署，PostgreSQL可靠 |
| **认证** | JWT | 管理后台权限控制 |

### 部署

| 服务 | 选择 | 费用 |
|------|------|------|
| **前端H5** | Vercel | 免费 |
| **域名** | 阿里云/腾讯云 | ~50元/年 |
| **云服务器** | 阿里云ECS/腾讯云 | ~100元/月起 |
| **数据库** | 随服务器或独立 | SQLite免费 |

## 数据库模型

### Product 商品表

```prisma
model Product {
  id          String   @id @default(cuid())
  name        String   # 商品名称
  brand       String   # 品牌
  model       String   # 型号
  size        String   # 尺码
  grade       String   # 成色等级 S/A/B/C
  price       Int      # 价格（分）
  status      String   @default("active") # active / inactive
  description String?  # 描述
  images      String   # JSON数组，图片URLs
  verifyCode  String?  # 验真编号，如 JZC-2026-000001
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
}
```

### Order 订单表（可选）

```prisma
model Order {
  id          String   @id @default(cuid())
  productId   String   # 商品ID
  buyerName   String   # 买家称呼
  buyerContact String  # 买家联系方式
  address     String?  # 收货地址
  status      String   @default("pending") # pending / shipped / completed
  remark      String?  # 备注
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
}
```

## API 设计

### 商品 API

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | /api/products | 获取商品列表 | 公开 |
| GET | /api/products/:id | 获取商品详情 | 公开 |
| POST | /api/products | 新增商品 | **管理员** |
| PUT | /api/products/:id | 更新商品 | **管理员** |
| DELETE | /api/products/:id | 删除商品 | **管理员** |

### 订单 API（可选）

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | /api/orders | 获取订单列表 | **管理员** |
| POST | /api/orders | 创建订单 | 公开 |
| PUT | /api/orders/:id | 更新订单状态 | **管理员** |

### API 响应格式

```json
{
  "success": true,
  "data": { ... },
  "message": "操作成功"
}
```

## 开发环境搭建

### 前置要求

- Node.js >= 18
- npm 或 yarn
- Git

### 步骤

```bash
# 1. 克隆项目
git clone <repo-url>
cd jianzhengcang

# 2. 安装前端依赖
cd frontend
npm install

# 3. 安装后端依赖
cd ../backend
npm install

# 4. 启动开发服务器
# 前端
cd frontend && npm run dev

# 后端
cd backend && npm run dev
```

## 部署流程

### 1. H5 前端部署到 Vercel

```bash
cd frontend
npm install -g vercel
vercel --prod
```

### 2. 后端部署到云服务器

```bash
# 1. 在服务器上安装 Node.js 18+
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 2. 拉取代码
git clone <repo-url>
cd jianzhengcang/backend
npm install --production
npm install -g pm2
pm2 start src/app.js --name jianzhengcang-api

# 3. 配置 Nginx 反向代理
```

### 3. 域名解析

```
A记录: @ -> 服务器IP
CNAME: www -> 你的域名
```

## 安全考虑

### 管理后台权限

1. JWT Token 认证
2. Token 过期时间：7天
3. 敏感操作需要验证 Token

### CORS 配置

```javascript
// 后端 CORS 配置
app.use(cors({
  origin: ['https://jianzhengcang.com', 'https://admin.jianzhengcang.com'],
  credentials: true
}));
```

### 敏感信息

- 数据库密码、JWT Secret 放在 `.env` 文件，不提交到 Git
- 使用 `dotenv` 加载环境变量
