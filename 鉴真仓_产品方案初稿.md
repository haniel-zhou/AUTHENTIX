## 1. 项目一句话定位

**鉴真仓是一个专注二手正品鞋的验真交易平台。**

平台不做综合二手交易，而是围绕二手鞋建立“入仓、验真、清洗、评级、建档、交易、售后”的可信流程。

## 4. 产品核心逻辑

### 4.1 买家问题

买家在二手鞋交易中最担心：

1. 买到假鞋；
2. 成色描述不可信；
3. 鞋子有气味、污渍、磨损；
4. 售后规则不清楚；
5. 商品图片和实物差异大。

平台提供的价值：

1. 平台验真；
2. 平台清洗；
3. 平台成色评级；
4. 平台建立验真档案；
5. 支持假货举报；
6. 卖家缴纳可退履约保证金；
7. 平台统一售后规则。

### 4.2 卖家问题

卖家在普通平台卖二手鞋时缺少信任背书。鉴真仓通过平台验真、清洗、评级、建档，提高买家信任，减少真假与成色争议。

## 9. 卖家可退履约保证金制度

### 9.1 制度定位

卖家保证金名称：

**卖家履约保证金**

性质：

- 不是入驻费；
- 不是服务费；
- 不是不可退押金；
- 是用于保障正品承诺、售后义务、纠纷处理和赔付责任的履约凭证。

### 9.2 金额建议

初始保证金建议：

**200-500 元**

后续可根据卖家等级、商品价格、历史交易、违规记录、售后表现动态调整。

### 9.3 缴纳后权限

卖家完成实名认证、资料审核并缴纳履约保证金后，才可以发布商品。

未缴纳保证金的用户：

- 可以浏览；
- 可以购买；
- 不能发布销售商品。

### 9.4 退还规则

卖家可以主动申请退还保证金。

申请后：

1. 平台立即下架其所有在售商品；
2. 平台冻结其卖鞋权限；
3. 卖家仍需处理历史订单、售后、纠纷、赔付；
4. 若无未完成事项，平台按原支付路径退还保证金；
5. 保证金退还完成后，卖家身份自动失效，只保留普通用户身份。

### 9.5 复卖规则

卖家退还保证金后，如需重新卖鞋：

1. 重新提交卖家申请；
2. 平台重新审核资料；
3. 审核通过后重新缴纳保证金；
4. 方可恢复卖鞋权限。

### 9.6 可扣除情形

平台只有在以下情形下，才可以按规则扣除全部或部分保证金：

- 经平台核实，卖家销售假鞋、仿鞋、高仿鞋或来源不明商品；
- 卖家虚假描述商品成色、品牌、型号、尺码或瑕疵；
- 卖家拒绝履行退款、退货、赔付或售后义务；
- 卖家恶意逃避纠纷处理；
- 卖家给平台、买家或第三方造成实际损失。

扣除前：

- 平台应说明理由；
- 提供申诉入口；
- 保留处理记录。

### 9.7 退保证金提示文案

```text
你正在申请退还卖家履约保证金。

提交申请后：
1. 你的所有在售商品将立即下架；
2. 你的卖鞋权限将被冻结；
3. 你仍需处理历史订单、售后纠纷和未完成赔付；
4. 审核通过后，保证金将按原支付路径退回；
5. 退还完成后，如需重新卖鞋，必须重新申请并再次缴纳保证金。

是否确认申请退还？
```

## 10. 成色评级体系

### 10.1 评级分级

建议使用：

- S级；
- A级；
- B级；
- C级；
- D级可作为内部低质级，不建议首页主推。

### 10.2 等级定义

#### S级：几乎全新

- 极少穿着痕迹；
- 鞋底轻微或无明显磨损；
- 鞋面无明显折痕；
- 内里干净；
- 无明显异味。

#### A级：轻微使用痕迹

- 有轻微折痕；
- 鞋底轻微磨损；
- 整体观感良好；
- 无明显破损；
- 无明显异味。

#### B级：正常使用痕迹

- 有正常折痕；
- 鞋底有可见磨损；
- 鞋面可能有轻微污渍或痕迹；
- 不影响正常穿着。

#### C级：明显磨损但可穿

- 鞋底磨损较明显；
- 鞋面有明显折痕、污渍或瑕疵；
- 仍可正常穿着；
- 需要在商品页明确展示问题。

#### D级：瑕疵较多

- 明显磨损；
- 可能有开胶、破损、内里损耗；
- 不建议首页展示；
- 若出售，必须低价并清楚标注。
