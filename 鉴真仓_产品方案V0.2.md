# 鉴真仓 · 产品方案 V0.2

> 版本：V0.2
> 日期：2026-04-16
> 状态：✅ 方案已确认，待开发

## 一句话定位

**鉴真仓是 Mattaniah 自营的二手正品球鞋店铺，用 H5+PWA 技术让用户可以添加到手机桌面，像 App 一样使用。**

---

## 与 V0.1 的核心区别

| 维度 | V0.1（旧） | V0.2（新版） |
|------|------------|--------------|
| **模式** | 平台型（多卖家+多买家） | **店铺型（Mattaniah自营卖货）** |
| **复杂度** | 高（含验真/保证金/举报） | **极简（展示+联系方式）** |
| **目标用户** | 模糊 | **Mattaniah自己 + 买家** |
| **技术路径** | 小程序+PC后台 | **H5(PWA) + PC后台** |
| **支付** | 在线支付 | **微信转账（极简版）** |
| **用户系统** | 需要 | **不需要** |

## 技术方案

### 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                    用户端（H5 + PWA）                    │
│                  域名: jianzhengcang.com                 │
│              (手机/电脑浏览器访问，可添加到桌面)            │
└─────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────┐
│                      后端 API                            │
│                 (Node.js 或 Python)                      │
└─────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────┐
│                     管理后台（PC）                        │
│              内部使用：商品管理 + 订单管理                 │
└─────────────────────────────────────────────────────────┘
```

### 技术选型

| 端 | 技术 | 说明 |
|----|------|------|
| **用户端 H5** | Vue 3 + Vite + TailwindCSS | 响应式，手机+电脑都能访问 |
| **PWA支持** | vite-plugin-pwa | 实现"添加到桌面"功能 |
| **后端** | Node.js（Express/Koa） 或 Python（FastAPI） | 待定 |
| **管理后台** | Vue 3 + Element Plus | PC端专用 |
| **部署** | Vercel（前端） + 任意云服务器（后端） | 成本低 |

### PWA 实现效果

```
用户用手机浏览器打开 jianzhengcang.com
        │
        ▼
   点击"添加到桌面"
        │
        ▼
  生成桌面图标
        │
        ▼
  点开像原生App一样全屏运行
```

## 极简MVP功能范围

### 用户端（H5）

#### 1. 首页
- 品牌展示（Logo + 主标语）
- 商品列表（6个示例商品）
- 核心信任入口（验真说明、评级说明）
- Footer（联系我们、用户协议入口）

#### 2. 商品详情页
- 商品图片
- 品牌 + 型号 + 尺码
- 成色等级 + 状态
- 价格
- 验真编号展示
- 联系方式（二维码/微信）
- 底部 Footer

#### 3. 联系方式页
- 微信二维码展示
- 或直接加微信沟通
- 简洁不复杂

#### 4. PWA 功能
- 添加到桌面提示
- 全屏运行
- 离线缓存（可选）

### 管理后台（PC）

#### 1. 商品管理
- 商品列表（展示/下架）
- 新增商品
- 编辑商品
- 删除商品

#### 2. 商品字段
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| name | string | 商品名称 |
| brand | string | 品牌（Nike/Adidas/...） |
| model | string | 型号 |
| size | string | 尺码 |
| grade | string | 成色等级（S/A/B/C） |
| price | number | 价格 |
| status | string | 状态（展示中/已下架） |
| images | string[] | 图片URL |
| verifyCode | string | 验真编号 |
| description | string | 描述 |
| createdAt | date | 创建时间 |

#### 3. 订单管理（可选）
- 订单列表
- 订单状态（待发货/已发货/已完成）
- 买家联系方式

### 等级定义

|------|------|
| **S级** | 几乎全新，无明显穿着痕迹 |
